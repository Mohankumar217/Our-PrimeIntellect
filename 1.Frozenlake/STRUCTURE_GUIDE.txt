2. MODULE DESCRIPTIONS
----------------------
- World/: Contains the "Ground Truth". It knows where the holes are but doesn't speak English.
- wrapper/: The Translator. Converts the World's grid state into natural language for the Agent, and parses the Agent's XML output back into game actions.
- agent/: The Player. Can be a Mock (random), a Real LLM (Gemini/Qwen), or updated implementations. It simply takes text in and outputs text.
- verifier/: The Judge. Independent logic to score performance based on the history of the episode.

3. ARCHITECTURAL STANDARD
-------------------------
This project follows the "Prime-Intellect Standard" for LLM Evals:
1. Separation of Concerns: Physics (World) != Interface (Wrapper) != Logic (Agent).
2. Statelessness: The Wrapper does not hold state; it only translates.
3. Parsability: Strict input/output formats (XML) to prevent hallucination issues.
